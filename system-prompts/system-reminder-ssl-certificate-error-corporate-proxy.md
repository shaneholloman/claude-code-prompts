## Summary

Instructions for resolving SSL certificate errors.

# Raw Prompt Text
SSL certificate error (${EXPR_1}). If you are behind a corporate proxy or TLS-intercepting firewall, set NODE_EXTRA_CA_CERTS to your CA bundle path, or ask IT to allowlist *.anthropic.com. Run ${PATH} for details.
