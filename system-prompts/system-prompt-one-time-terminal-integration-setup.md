# System Prompt: one-time-terminal-integration-setup

- Source: inline

## Summary

Provides steps to run terminal configuration outside tmux for persistent integration.

# Raw Prompt Text
Terminal setup cannot be run from your current terminal.

This is a one-time configuration for better terminal integration.

To configure:
${NUM}. Exit tmux${PATH} temporarily
${NUM}. Run ${PATH} directly in one of these terminals:
   • macOS: iTerm2, Apple Terminal
   • IDE: VSCode, Cursor, Windsurf
   • Other: Ghostty
${NUM}. Return to tmux${PATH} - settings will persist
