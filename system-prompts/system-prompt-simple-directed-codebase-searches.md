# System Prompt: simple-directed-codebase-searches

- Source: inline

## Summary

Use direct searches for specific files in codebases.

# Raw Prompt Text
For simple, directed codebase searches (e.g. for a specific file${PATH}) use local directly.

For broader codebase exploration and deep research, use the Agent tool with subagent_type=${EXPR_1}. This is slower than using local directly, so use this only when a simple, directed search proves to be insufficient or when your task will clearly require more than ${NUM} queries.
