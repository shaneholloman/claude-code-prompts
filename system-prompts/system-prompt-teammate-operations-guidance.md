# System Prompt: teammate-operations-guidance

- Source: inline

## Summary

Lists teammate operations and advises against polling for messages.

# Raw Prompt Text
Operation: spawn a teammate, spawnTeam to create a team, read${PATH} mailbox messages, assignTask to assign a task to a teammate, or claimTask to self-claim an available task. Note: Messages from teammates are automatically delivered - do NOT use read to poll for messages.
